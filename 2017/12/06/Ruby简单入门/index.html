字符串

"Hello, world" 是一个字符串。在 Ruby 中，字符串同样也是一个对象，被称为 String 对象或者字符串对象。字符串、数值、时间等各种数据，在 Ruby 中都是对象。

转义符 \

想在字符串里嵌入特殊符号时，就会用到 \。在 Hello, world 程序中出现的 \n 是换行符。又比如双引号是表示字符串开始和结束的符号，如果希望字符串里包含双引号，程序要写成 \"；如果字符串中需要包含 \，程序要写成 \\。

‘ ‘ 与 “ “

除了可以使用 " "表示字符串外，也可以使用 ' '，在单引号里，像 \n 这样的特殊字符不经过转义，程序会原封不动地直接输出。但也有例外，例如在字符串里想嵌入 \ 与单引号时，还是需要在之前加上 \。

p 方法

无论使用 print 方法还是 puts 方法，输出数值 100 和字符串 "100" 时，结果都只是单纯的 100。这样我们无法直接判断输出的结果到底是属于哪个对象。这种情况下，p 方法可以很好地解决这个问题。使用 p 方法时，数值结果和字符串结果会以不同的形式输出。

另外，使用 p 方法时，换行符 \n、制表符 \t 等特殊字符不会转义。

如果只是需要输出程序执行的结果、信息等，可以选择 print 方法，如果想确认程序的执行情况，则可选择 p 方法。原则上，p 方法是提供给编程者使用的。

使用中文的注意事项

设定编码

编码设定不当会导致中文输出错误、乱码等情况，Ruby 的编码方式，是通过在程序的首行代码添加注释 # encoding: 编码方式 来指定的(编码的规则称为 encoding)。这个注释被称为魔法注释(magic comment)。

下表是各平台常用的编码方式。平台有多个常用编码方式时，请按照实际情况选择适合的编码方式。

平台 编码方式
Windows GBK(或者 GB 2312)
Mac OS X UTF-8
Unix UTF-8

若没指定魔法注释，Ruby 会默认使用 UTF-8 编码方式。因此如果希望代码采用 UTF-8 的编码方式时，可省略魔法注释。

使用“-E 编码方式”

数值

数值也是以数值对象的形式存在的。 Ruby 中整数和小 数(浮点数)的处理方式都很自然，可以直接表示或进行四则运算。

使用数学函数，如平方根、sin 和 cos 等三角函数、指数函数等。使用数学函数时，必须在函数前加上 Math.。不想在函数前加 Math. 时，则需要 include Math 这段代码。

变量

Ruby 中变量除了通常的用法，还可以这样:

"表面积= #{area}\n" 整体为一个字符串。#{area} 这样的写法表示把 area 的值嵌入到字符串中。在字符串中使用 #{变量名} 这样的写法，可以把通过计算后得到的值嵌入到字符串中。输出结果中除了可以嵌入变量名，也可以嵌入 "表面积=#{(x*y + y*z + z*x) * 2}\n" 这样的计算公式，得到的输出结果是一样的。

在散列中，一般以字符串或者符号作为键，来保存对应的对象。

符号

符号与字符串对象很相似，符号也是对象，一般作为名称标签使用，表示方法等的对象的名称。可以将符号简单理解为轻量级的字符串。

通过在标识符的开头加上 : 来创建符号:

符号能实现的功能，大部分字符串也能实现。但在像散列的键这样只是单纯判断“是否相等”的处理中，使用符号会比字符串更有效率，因此在实际编程中也会时常用到符号。

符号与字符串可以互相转换。对符号使用 to_s 方法，则可以得到对应的字符串。反之，对字符串使用 to_sym 方法，则可以得到对应的符号。
